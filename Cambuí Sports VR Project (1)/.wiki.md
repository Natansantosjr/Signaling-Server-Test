# Project Summary
O projeto **Cambuí Sports VR** é uma plataforma de transmissão ao vivo de eventos esportivos em um ambiente de realidade virtual. Ele oferece um painel web que permite o controle e monitoramento de transmissões imersivas, com funcionalidades como visualização de múltiplas câmeras, sistema de replay instantâneo, analytics em tempo real, e integração com blockchain para gerenciamento de wallets e NFTs.

# Project Module Description
- **Dashboard Layout**: Gerencia a navegação lateral e a disposição responsiva do conteúdo.
- **Live Streaming**: Exibe um grid de câmeras com opções de qualidade e métricas de transmissão.
- **Replay System**: Permite a reprodução de eventos gravados com controles de velocidade.
- **Events**: Lista eventos ao vivo e agendados, com informações sobre as partidas.
- **Analytics**: Apresenta métricas em tempo real, distribuição de câmeras e gráficos de audiência.
- **Wallet/Blockchain**: Exibe informações sobre saldo, NFTs e status da blockchain.
- **Infrastructure**: Monitora a saúde do cluster LiveKit, codificadores e CDN.

# File Description Inventory
- **backend**: Contém a lógica de servidor, modelos de dados, e rotas para a API.
- **frontend**: Inclui componentes React, páginas, e estilos para a interface do usuário.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Shadcn/ui
- **Backend**: Atoms Cloud (Auth, Database, File Storage)
- **Database**: PostgreSQL via Atoms Cloud entities
- **Protocol**: WebRTC para transmissão em tempo real

# Usage
1. **Instalação de Dependências**: Navegue para o diretório `frontend` e execute `pnpm install`.
2. **Compilação do Projeto**: Execute `pnpm run build` no diretório `frontend` para compilar o projeto.
3. **Execução do Projeto**: Após a compilação, inicie o servidor de desenvolvimento com `pnpm run start`.
